Security Infrastructure

Cryptographic certainty without backchannels.

  • End-to-End Encryption

    ECIES hybrid encryption ensures data stays unreadable from the offline token holder until it reaches the authoritative Bank HSM.

  • ECDSA Signatures

    Every transaction intent is cryptographically signed using SECP256R1 signatures directly inside the device's hardware enclave.

  • Hardware Attestation

    Integrates dynamic TPM validation, verifying device integrity prior to permitting offline spending allowance.

  • Replay Attack Resistance

    Stateful on-ledger checks block re-entry by verifying monotonically increasing nonces.

  • Byzantine Relay Resistance

    Protects the host network from malicious relay behavior by enforcing payload expiration bounds and hop limits.
